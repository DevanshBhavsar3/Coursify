# Coursify - A Course Selling App

## Technologies:

### Backend:

- Express
- Json Web Token
- Bcryt
- Mongoose
- Zod

### Frontend:

- React
- React Router
- Axios
- Tailwind CSS

## How to Run Locally:

- Clone this Repository using git command-line on your system.

  ```
  git clone https://github.com/DevanshBhavsar3/Coursify.git
  ```

- Start the server.

  ```
  cd server
  npm i
  node index.js
  ```

- Copy .env.

  ```
    cp .env.example .env
  ```

- Start the frontend.

  ```
  cd client
  npm i
  npm run dev
  ```
